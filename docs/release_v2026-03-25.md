# Investment Hub Release `v2026.03.25`

Release date: `2026-03-25`

## Version Baseline

- Backend API version: `investment-backend v2026.03.25`
- Frontend dashboard version: `investment-frontend v2026.03.25`
- Runtime scheduler pack: `scheduler-v2`

## Included Changes

1. Opportunity pool and scoring
- Core index pool sync and scoring APIs.
- Drill-down stock analysis panel.

2. Win-rate first dashboard
- Removed low-value content modules from home/research views.
- Added win-rate console, setup stability ranking, signal replay, discipline checklist.

3. Preopen strategy module
- Added preopen strategy generation API:
  - `GET /investment/api/strategy/preopen`
  - `GET /investment/api/strategy/preopen/latest`
- Added Tonghuashun quant readiness plan output (with compliance constraints).

4. Minute-level runtime refresh
- Added asynchronous runtime refresh API:
  - `POST /investment/api/runtime/minute-refresh`
- Added minute task + preopen task scripts for Windows Scheduler.

5. Bailian daily intelligence integration
- Added Bailian OpenAI-compatible intelligence service and APIs:
  - `GET /investment/api/intelligence/brief/latest`
  - `POST /investment/api/intelligence/brief/refresh`
- Added scheduled daily intelligence generation tasks (preopen + postclose).
