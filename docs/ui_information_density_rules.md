# UI Information Density Rules

## 首页

首页仍然是决策中心，不是数据仓库浏览器。

## 高优先级结论必须一键可见

- 原始数据
- 原始来源
- 本地归档
- 最近报告
- 引用片段
- 缺失确认项

## 禁止

- 重复平铺同一 thesis。
- sample/fallback 混进可执行前台。
- 空白关键字段。
- 只显示分数、不显示原始数据。
- 大段重复解释文案。

## 列表过滤

所有核心列表支持：

- 来源过滤
- 市场过滤
- 赛道过滤
- family 过滤
- live-only 过滤
- archived-only 过滤

## 时间

- 所有时间统一 timezone-aware。
- 已过期检查点显著提醒。
- 自动滚动到下一个合法时点。
