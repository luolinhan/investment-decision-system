# Opportunity Universe Registry

V3 不再把机会宇宙写死为五条赛道。系统使用 registry 管理 sector、theme、entity、instrument、mapping、model、thesis 和 event template。

## Registry

- `sector_registry`
- `theme_registry`
- `entity_registry`
- `instrument_registry`
- `mapping_registry`
- `model_registry`
- `thesis_registry`
- `event_template_registry`

## 至少可用的 12 个主题模板

1. AI 算力基础设施
2. AI 应用 / 软件
3. 创新药
4. CRO / CXO / 科研服务
5. 半导体设备
6. 半导体材料 / 代工 / 封测 / 设计
7. 光伏
8. 电网 / 储能 / 电力设备
9. 锂电 / 电动车
10. 机器人 / 自动化
11. 军工 / 卫星 / 航空航天
12. 黄金 / 铜 / 铝 / 稀土等有色

后续扩展到券商、保险、银行红利、煤炭油气电力核电、航运港口物流、互联网平台、消费电子、工程机械、旅游免税、地产链、猪周期、化工农化等。

## 每个模板必须定义

- `sector_id`
- 中文名 / 英文名
- lead assets
- bridge assets
- local assets
- proxy assets
- upstream nodes
- downstream nodes
- key metrics
- key events
- default invalidation rules
- replay horizons
- source requirements
